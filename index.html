二、我们如何收集和使用您的个人信息

1. 基础运行与安全

为保障本游戏正常运行、识别异常、优化体验和完成网络请求，我们或集成的第三方 SDK 可能处理设备型号、系统版本、网络类型、IP 地址、应用包名、应用版本、运行状态、崩溃和性能数据等信息。

2. TapTap 登录

当您选择 TapTap 登录时，我们会通过 TapSDK 获取 TapTap 授权结果，并可能使用您的 TapTap openId、unionId、昵称、头像等公开资料，用于识别您的游戏账号、显示登录状态、同步存档和保障账号安全。如您拒绝登录，部分依赖账号的功能可能无法使用。

3. 实名认证与防沉迷

为履行网络游戏实名制和未成年人保护相关要求，本游戏接入 TapTap 合规认证能力。实名认证由 TapTap 相关服务完成，我们通常仅接收认证状态、是否允许进入游戏、年龄段或限制状态等结果信息，不主动保存您的身份证件号码。

4. 云存档与本地存档

本游戏会在您的设备本地保存剧情进度、选项记录、证据收集、结局状态、音量设置等游玩数据。登录 TapTap 后，您可使用云存档功能，云端可能保存上述存档内容及存档时间，用于跨设备恢复或继续游玩。

5. 激励广告

当您点击“看广告解锁”等入口时，本游戏会调用 TapADN/Dirichlet 广告 SDK 展示激励广告。广告 SDK 可能处理设备信息、网络信息、应用信息、广告展示、点击、观看完成、转化、反作弊和性能数据，用于广告投放、统计分析、广告归因和奖励发放。

6. 动态背景、手机震动与设备能力

本游戏可能读取设备旋转矢量、加速度等传感器数据，用于全景或动态背景的视角跟随效果。该类传感器数据仅用于本地画面表现，不用于识别您的身份。来消息提醒等场景可能调用设备震动能力，用于增强游戏反馈。

7. 分享、评价、成就与更新唤起

当您主动使用 TapTap 分享、打开评价、查看或解锁成就、检查更新等功能时，相关 TapSDK 模块可能处理 TapTap 账号标识、设备信息、网络状态、应用包名、应用版本、已安装 TapTap 客户端信息、成就 ID、成就进度、分享内容等必要信息。

五、我们如何保存您的个人信息

我们将在实现本政策所述目的所需的最短期限内保存您的个人信息和游戏存档。法律法规另有规定的，从其规定。超出必要期限后，我们会删除或匿名化处理相关信息。

本游戏在中国境内运营过程中收集和产生的个人信息原则上存储在中国境内。如因您主动使用 TapTap 或第三方服务导致信息由第三方处理，第三方将按其隐私政策说明存储地点、期限和保护措施。

六、我们如何共享、转让、公开披露您的个人信息

我们不会出售您的个人信息。为实现本政策所述功能，我们可能向 TapSDK、TapADN SDK 等第三方服务提供方提供必要信息，或由第三方 SDK 按其功能自动采集必要信息。我们会要求第三方按照法律法规和协议约定处理您的个人信息。

除非取得您的单独同意、法律法规要求、行政司法机关依法提出要求，或为维护您、他人和公共安全所必需，我们不会公开披露您的个人信息。若发生合并、分立、收购、资产转让等情形并涉及个人信息转让，我们会要求新的持有方继续受本政策约束，否则将要求其重新取得您的授权同意。

七、您如何管理个人信息

您可以通过设备系统设置管理网络、安装包、震动等权限。您也可以通过 TapTap 账号体系管理 TapTap 账号资料、授权、实名状态、云存档等由 TapTap 提供的相关数据能力。

如您希望访问、更正、删除您的个人信息，撤回授权同意，注销账号，或投诉举报，请通过本政策载明的联系邮箱与我们联系。为保障安全，我们可能会要求您提供必要信息以验证身份。我们将在收到并验证您的请求后十五个工作日内处理或答复。

请您理解，当您撤回授权或删除必要信息后，与该信息对应的功能可能无法继续使用，但不会影响此前基于您授权已经完成的个人信息处理活动。

八、未成年人保护

我们重视未成年人个人信息保护。若您未满十八周岁，请在监护人指导下阅读并使用本游戏；若您未满十四周岁，请在监护人同意后使用本游戏。我们接入 TapTap 合规认证与防沉迷能力，用于识别未成年人游戏时长、登录限制和相关合规要求。

如监护人发现未成年人未经同意使用本游戏或提供个人信息，请通过本政策载明的联系方式与我们联系，我们会依法处理。

九、本政策如何更新

本政策可能因功能变化、SDK 调整、法律法规更新或运营需要而更新。若发生重大变更，我们会在游戏内、下载页面或其他适当位置进行显著提示。未经您明确同意，我们不会削减您依据本政策享有的权利。
